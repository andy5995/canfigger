Detailed Description

Header file for the Canfigger configuration parser.

This library provides functionality to parse configuration files, creating and managing a linked list representation of keys, values, and attributes found within the file. It supports operations for parsing, traversing, and freeing the created lists.

Part of canfigger (https://github.com/andy5995/canfigger).
